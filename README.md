# jianghu_h5
江湖前端

## Start

```shell
cnpm install
npm start
```
## IMPORTANT

提交代码前必须执行: `npm run lint`

## 协作方式

1. fork 到自己的仓库
2. 克隆
3. developing...
4. commit
5. pull upstream
6. push
7. pull request

## 与上游保持更新

```shell
git remote add upstream git@github.com:ldsn/jianghu_h5.git
git pull upstream
```
